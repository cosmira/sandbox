# Sandbox Sync Benchmark

Этот benchmark тестирует производительность синхронизации данных между активной таблицей и sandbox таблицей.

## Установка

Пакет `dragon-code/benchmark` уже установлен как dev-зависимость:

```bash
composer require dragon-code/benchmark --dev
```

## Запуск Benchmark

Результаты будут выведены в консоль в виде таблицы.

```bash
php vendor/bin/testbench sandbox:benchmark --count=100000
```

Параметры:
- `--count=100000` - количество записей для синхронизации (по умолчанию 100,000)

## Результаты

Benchmark измеряет время выполнения двух операций:

1. **Active > Sandbox** - синхронизация из активной таблицы в sandbox
   - Удаление orphan записей
   - Вставка недостающих записей
   
2. **Sandbox > Active** - синхронизация из sandbox в активную таблицу
   - Удаление orphan записей
   - Вставка недостающих записей

### Пример вывода для 100,000 записей:

```
+-------+--------------------------------------------+--------------------------------------------+
| #     | Active > Sandbox (insert + delete orphans) | Sandbox > Active (insert + delete orphans) |
+-------+--------------------------------------------+--------------------------------------------+
| min   | 157.041042 ms - 0 bytes                    | 208.120625 ms - 0 bytes                    |
| max   | 157.041042 ms - 0 bytes                    | 208.120625 ms - 0 bytes                    |
| avg   | 157.041042 ms - 0 bytes                    | 208.120625 ms - 0 bytes                    |
| total | 157.041042 ms - 0 bytes                    | 208.120625 ms - 0 bytes                    |
+-------+--------------------------------------------+--------------------------------------------+
| order | 1                                          | 2                                          |
+-------+--------------------------------------------+--------------------------------------------+
```

## Заметки о производительности

- Benchmark использует SQLite in-memory для тестирования
- Время синхронизации зависит от размера набора данных и количества изменений
- Для реальной базы данных результаты могут отличаться

## Файлы

- `src/Commands/BenchmarkSyncCommand.php` - консольная команда для запуска benchmark'а
